功能详解

每一个细节都经过打磨

从底层架构到交互细节，Wisector LearnLM 追求生产级的工程质量。

智能对话系统

支持 OpenAI、Anthropic、Gemini 多家模型，可配置自定义端点、API Key、上下文窗口与定价。

  • 推理强度 5 档可调（Low / Medium / High / xHigh / Max）
  • 网络访问开关，支持实时联网搜索
  • 询问 / 编辑双模式切换
  • 附件上传：图片、文件、引用笔记

AI 视频创作

5 步向导式流程，从选题到合成全自动，适合快速制作教学微课。

  • 支持单选课本或「课本 + 题目」组合模式
  • 多模态主模型驱动幻灯片生成与圈点动画
  • TTS 配音：speech-2.8-hd、Gemini TTS
  • 图片生成：4 种风格（插画 / 线稿 / 写实 / 卡通）

课本管理

独立窗口选择课本，支持编辑、删除、按学科筛选，题目严格按课本匹配。

  • 中央弹出式独立选择窗口（Portal 渲染）
  • 多版本教材：人教版、教科版、外研版、浙教版
  • 支持搜索、筛选、新建
  • 题目按课本名称严格筛选

数据管理

上传试卷后，AI 自动识别题目内容、题型（选择 / 填空 / 解答 / 判断）、难度与参考答案。

  • 多模态识别模型：Claude Opus 4.8、Gemini 3.1 Pro、GPT-5.5
  • 题目按合集（Collection）组织
  • LaTeX 公式渲染（行内 + 块级）
  • 可视化编辑题目与课本

预设模型库

内置 10+ 主流模型预设，一键填入 endpoint、上下文窗口、输入模态等配置。

  • 对话模型：Claude Fable 5、Opus 4.8、GPT-5.5、Gemini 3.1 Pro
  • 生图模型：GPT Image 2、Nano Banana 2、Seedream 4
  • TTS 模型：speech-2.8-hd、Gemini TTS Preview
  • 支持自定义模型、定价配置（CNY / USD）

移动端适配

采用 mobile-first 设计，全屏弹窗、抽屉式侧边栏、弹性网格布局。

  • 移动端设置弹窗全屏（dvh 动态视口）
  • 横向滚动 Tab 导航
  • Framer Motion 流畅过渡动画
  • 完整触摸友好的交互设计

项目结构

清晰的代码组织

采用 Next.js App Router 约定，组件化拆分，关注点分离。

app/ # App Router 页面 data/ page.tsx # 数据管理页（合集/题目/上传识别） video/ page.tsx # 视频创作页（向导 + 模型选择） page.tsx # 首页（AI 对话） layout.tsx # 根布局（含 KaTeX CSS） globals.css # 全局样式 components/ # 通用组件 ChatArea.tsx # 对话区（输入框 / 工具栏 / 快捷操作） Sidebar.tsx # 侧边栏（导航 / 笔记 / bottomSlot） TextbookSelector.tsx # 课本选择器（Portal 独立窗口） ModelSelector.tsx # 对话模型选择下拉 ModelsPanel.tsx # 模型管理面板（Tab 分类） SettingsModal.tsx # 设置弹窗（移动端全屏） Math.tsx # LaTeX 块级公式组件 LaTeXText.tsx # LaTeX 行内公式解析 Logo.tsx # 品牌 Logo lib/ # 工具与类型 api.ts # 后端 API 调用封装 types.ts # 共享类型定义 useModelConfigs.ts # 模型配置 Hook server/ # 后端服务 package.json src/ public/ # 静态资源 package.json # 项目配置 next.config.ts # Next.js 配置 tailwind.config.ts # Tailwind + Notion 设计系统 tsconfig.json # TypeScript 配置
快速开始

三步运行项目

克隆仓库、安装依赖、启动开发服务器。

1

克隆仓库

从 GitHub 克隆项目到本地

$ git clone https://github.com/modemw3294/Wisector-LearnLM-Alpha.git
2

安装依赖

使用 npm 或 yarn / pnpm 安装

$ cd Wisector-LearnLM-Alpha
$ npm install
3

启动开发

运行开发服务器，访问 localhost

$ npm run dev
# 访问 http://localhost:3000
